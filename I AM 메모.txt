# I AM — Gemini Canvas로 내가 고른 주제를 소개하는 나만의 공간

> **프로젝트명:** I AM (아이엠)
>
> **한 줄 정의:** 내가 좋아하는 주제 (나 자신, 포트폴리오, 취미, 반려동물 등) 를 매력적으로 소개하는 단일 페이지 웹 서비스
>
> **목적:** 보여주고 싶은 내용을 AI 의 도움으로 화면으로 구현하며, 개발의 재미와 성취감을 느끼기
>
> **트랙:** HTML5 + CSS + Vanilla JS 단일 파일로, Gemini Canvas 에 화면 구성을 요청하고 디자인을 직접 수정해보기
>
> **목표:** 시간 안에 바닐라 JS 기반 1 페이지 (SPA) 소개 사이트 기본형 완성
>
> **사용 기술:** Vanilla JS (순수 JS) / CSS / HTML


[페르소나]
너는 브랜딩 기획자야. 웹사이트 제작 전에 컨셉을 정리하도록 도와줘. 코드는 아직 쓰지 마.

[정보]
- 소개 대상: {{ 나 자신 / 포트폴리오 / 취미 / 반려동물 / 큐레이션 등 }}
- 자유롭게 적은 내용: {{ 편하게 나열 }}

[요청]
아래 6가지를 하나씩 질문하고, 답변이 끝나면 표로 정리해줘.
1) 캐치프레이즈 후보 3개
2) 매력 포인트 3~5개 (구체적으로)
3) 타겟 방문자
4) 스토리텔링 흐름
5) 무드/톤 (형용사 3개)
6) 메인 컬러 + 폰트
